fmt - стандартный пакет базовых команд для ввода\вывода

примитивы:
- числове типы (int, float)
- строчный тип (string)
- логический тип (bool)

- var - обьявляет создание новой переменной, выделяет память, назначает неизменяемый тип, дает значение по умолчанию, если не указано
- но все таки короткая запись := без явного указания типа - идеоматична для go
- код не запустится если есть неиспользованная переменная

Строки:
В го строки это массив значений типа byte
строки неизменяемы
к эелементам можно обращаться по индексу

методы: 
- len() - длина строки -  есть нюанс, длина в байтах не всегда корректна, тк го можно работать с Unicode в которой не все символы умещаются в один байт
- utf8.RuneCountInString(s) - специальная команда для длины строки из пакета utf8 для Unicode
- если заключить в `` то все будет отформатированно как если бы внутри были управляющие последовательности


Пользовательские типы:
type Fruit string
var fruit Fruit
Для пользовательских типов можно определять методы (а для втроенных нельзя)

Приведение типов:
Чтобы привести один тип к другому - type(variable)

абсолютно для всех типов можно юзать алиасы



# Переменные
обьявление ну и инициализация
var name type = expression
без указания типа (если сразу инициализация)
var name = expression
короткая нотация
name := expression

reflect.TypeOf() - возвращает тип

Эквивалентные литералы:
1000
1000.0
1_000 // можно разделять части числа символом '_' для удобства восприятия
01750 // восьмеричное представление, начинается с 0
0x3e8 // шестнадцатеричное представление
0b001111101000 // бинарное представление

# Константы
const - определяет именованную константу
Именованные константы позволяют изменять значение только в одном месте кода

Для констант типизация ослаблена - например можно смешивать разные числовые типы
Но если константа типизирована - тогда типизация для нее будет сильной

Код можно запускать даже с неиспользуемой константой

Для задачи констант, суть которых не в самих значениях, а в отличиях от других констант - значению присваивается специальное слово iota.
Оно создает автоматически инкремент, начиная с 0
